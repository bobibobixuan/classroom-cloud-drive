# 更新记录

## 2026-04-24

### 本轮重点

- 注册流程改为白名单制，并补齐实名账号生成逻辑。
- 管理后台新增白名单导入子块，支持导入、状态流转和历史手机号匹配。
- 聊天抽屉继续做了布局与交互修复，重点解决发送区可见性和消息定位体验。

### 注册与实名体系

- 新增 registration_whitelist 白名单表及导入接口，支持 .txt / .csv 名单导入。
- 白名单导入支持“手机号,真实姓名”和“真实姓名,手机号”两种列顺序。
- 注册成功后会消费手机号名额，并将最终账号名生成成 昵称_真实姓名。
- 历史已注册用户会按手机号自动回填到白名单 registered 状态，不需要重新注册。
- 改名逻辑已收紧为只修改昵称部分，实名后缀保留。

### 前端身份与展示

- /api/me、登录响应、管理员用户列表和聊天消息都已补充 real_name 字段。
- 侧边栏、顶部工作区、管理员用户列表和聊天发言人支持“昵称 + 浅色实名”显示。
- 管理员用户页内部拆出“用户目录 / 注册白名单”两个收纳子块，避免继续纵向堆叠。

### 登录与聊天体验

- 登录页和注册页增加了常驻表单提示，不再只依赖 toast。
- 密码错误、账号不存在、账号冻结、字段缺失和注册成功后的登录指引都会在表单内直接显示。
- 聊天抽屉新增更稳定的自动滚动、重新打开定位到底部、回到最新消息按钮。
- 桌面端聊天抽屉挂载方式已改为固定视口层，发送区打开后即可见，不再跟着整页滚动到底部。

### 数据与兼容修复

- 启动迁移和名单导入时都会执行白名单与 users.phone 的回填对齐。
- 聊天消息读取会直接关联白名单实名，减少前端额外推断。
- 当前数据库已补做一次手机号回填，已匹配的历史账号可以立即显示实名。

## 2026-04-23

### 本轮重点

- 项目说明文档改写为当前真实状态，补充了云盘与仓库双系统说明。
- 新增并完善 Git 忽略规则，避免虚拟环境、数据库、仓库存储和运行时文件误入版本库。
- 整理了最近一轮关键功能、联调修复和部署注意事项。
- 管理后台从单页堆叠改为按分区路由切换，并把日志独立成单独菜单。

### 功能补全

- 新增仓库大厅，支持浏览公开仓库。
- 私有仓库支持邀请协作者共同维护。
- 仓库支持加入维护申请、撤回申请、管理员审核、协作者退出。
- 仓库支持公告、文件管理、仓库活动日志。
- 新增通知中心，支持未读红点和业务跳转。
- 管理后台支持用户配额、角色、冻结、密码重置、分享治理、回收站和审计日志。

### 权限与身份修复

- 修正登录后超级管理员被错误识别为子管理员的问题。
- /api/drive 已返回完整身份字段，包括 role、is_super_admin、admin_scopes、quota 等。
- 前端身份同步逻辑改为按字段合并，避免局部响应把超管身份覆盖掉。
- bobibobixuan 自动保持为超级管理员。
- 当创建或提升子管理员时，如果没有显式传入权限范围，后端会自动补齐默认管理范围。

### 管理后台结构调整

- 侧边栏已拆分为“管理员配置”和“操作日志”两个管理员菜单。
- 管理员配置内部改为按子页面切换，不再把用户、仓库、公开链接、回收站全部堆在同一长页。
- 新增并启用以下后台路由：
	- /admin/users
	- /admin/repos
	- /admin/shares
	- /admin/recycle
	- /admin/logs

### 仓库与容量策略

- 单仓库容量限制已提升到 1 GB。
- 云盘与仓库明确分离：云盘用于个人文件，仓库用于多人协作维护。
- 仓库支持公开与私有两种模式。

### 联调与代理修复

- [main.go](main.go) 现在会在未设置 CCD_API_BASE_URL 时优先探测本地 http://127.0.0.1:4321。
- 本地后端可达时，Go 代理自动进入本地联调模式。
- 本地后端不可达时，仍会回退到远端 API，保持机房中转部署能力。

### UI 修复

- 通知消息气泡不再被主页容器裁切。
- 通知面板已提升为高层级 fixed 浮层。
- 首页和工作台布局已为聊天栏与通知层预留更稳定的显示空间。
- 管理后台分区导航已简化为单屏切换，避免内容持续下堆。

### 已验证结果

- Go 编译通过：go build .
- Windows 可执行文件已成功打包：pan.exe。
- 本地 FastAPI 后端可正常启动在 4321。
- Go 代理日志已确认能够识别并使用本地后端。
